🌦️ Weather App
A sleek and simple Weather Application that fetches real-time weather data for any city or location worldwide!
Built using HTML, CSS, and JavaScript — no frameworks, pure web magic.


🚀 Features
🌍 Get current weather of any location on Earth.

🔎 Search by city name.

📡 Uses OpenWeatherMap API for accurate and real-time data.

📱 Fully responsive design for desktop and mobile.

☁️ Displays:

Temperature

Weather description

Humidity

Wind speed

Weather icons


💻 Tech Stack
HTML5 — Structure

CSS3 — Styling

JavaScript (ES6) — Logic & API handling

OpenWeatherMap API — Data provider

⚙️ Usage
Enter a city name into the search box.

Hit Search or press Enter.

Instantly view the real-time weather report for the selected location!
